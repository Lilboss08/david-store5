🚀 David Store - Quick Start Guide

Your complete e-commerce solution with Nigerian delivery support

1 Start the Backend Server

First, make sure your backend server is running:

cd backend npm start # or node index.js

The backend should be running on http://localhost:4000

2 Setup Admin Account

To manage products, you need an admin account. There are two easy ways:

Option A: Use Admin Setup Page (Recommended)

  1. Open admin-setup.html in your browser
  2. The page will automatically create the admin account
  3. You'll see confirmation when it's done

Option B: Use Command Line

cd backend node create-admin.js

4 Features Working

  • Product Management: Add, update, delete products (admin only)
  • Search: Search products by name or keywords
  • Categories: Filter products by category
  • Shopping Cart: Add products, manage quantities
  • Nigerian Delivery: Complete state and LGA selection
  • Delivery Fees: Automatic calculation based on location
  • User Authentication: Login, registration, admin access
  • Email Verification: Account verification system

6 Troubleshooting

Products Not Loading?

  • Check if backend server is running on port 4000
  • Check browser console for errors
  • Run comprehensive test to diagnose issues

Admin Functions Not Working?

  • Make sure you're logged in as admin
  • Check admin-setup.html to ensure admin account exists
  • Use debug-auth.html to check authentication status

LGAs Not Showing?

  • Check internet connection (needed for LGA API)
  • Falls back to hardcoded data if API unavailable
  • Try selecting a different state first

8 Technical Details

Technology Stack

  • Frontend: HTML5, CSS3, Vanilla JavaScript
  • Backend: Node.js, Express.js
  • Database: MongoDB
  • Authentication: JWT (JSON Web Tokens)
  • Styling: Custom CSS with responsive design

API Endpoints

  • Products: GET/POST /api/v1/products
  • Product: GET/PATCH/DELETE /api/v1/products/:id
  • Search: GET /api/v1/products/search?q=query
  • States: GET /api/v1/location/states
  • LGAs: GET /api/v1/location/lgas/:state
  • Auth: POST /api/v1/user/login
  • Register: POST /api/v1/user/register

Free LGA API

Uses https://nga-states-lga.onrender.com/api/{state} for Nigerian Local Government Areas
